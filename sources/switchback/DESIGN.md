# Switchback — Design brief

A rugged, condensed slab display in a single weight. Vintage wood-type
and western-poster lineage; meant for headlines, posters, badges, and
trail-distance signs.

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 720                        |
| x-height            | 520                        |
| Ascender            | 800                        |
| Descender           | -180                       |
| Default sidebearing | 50                         |
| Condense factor     | 0.78                       |
| Stroke              | 140 (Bold-class)           |
| Serif length        | ~2.4 × stroke              |
| Serif thickness     | ~0.7 × stroke (min 50)     |

Optimized for all-caps display use; lowercase included but the typeface
shines uppercase.
