# Campfire Script — Design brief

A casual brush script in a single weight. Inspired by hand-painted
park signs, ranger-station chalkboards, and WPA-era poster lettering.

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 700                        |
| x-height            | 500                        |
| Ascender            | 820                        |
| Descender           | -220                       |
| Slant               | 16°                        |
| Default sidebearing | 40                         |

## Status

v0.0.1 ships a sheared sans skeleton — true script connections,
swashy capitals, and the hand-painted texture that defines the genre
are TODO. Treat this family as an early placeholder.
