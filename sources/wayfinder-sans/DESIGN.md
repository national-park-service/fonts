# Wayfinder Sans — Design brief

A humanist sans-serif for body text, UI, and signage. Designed to share
metrics and rhythm with **Wayfinder Serif** so the two pair cleanly.

## Inspiration

Vintage NPS / WPA park signage; Frutiger-family humanist sans-serifs;
the open, generous apertures of typefaces designed for distance reading.

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 700                        |
| x-height            | 500                        |
| Ascender            | 800                        |
| Descender           | -200                       |
| Default sidebearing | 60                         |
