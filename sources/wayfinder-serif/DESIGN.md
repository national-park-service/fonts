# Wayfinder Serif — Design brief

A slab/transitional serif companion to Wayfinder Sans, designed for
long-form reading at every weight from Light through Black.

## Inspiration

Roadway-style slab serifs used in U.S. public-lands wayfinding; the
sturdy, low-contrast NPS Rawlinson Roadway tradition; transitional
text faces (Mercury, Lyon Text) for the lighter end.

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 700                        |
| x-height            | 500                        |
| Ascender            | 800                        |
| Descender           | -200                       |
| Default sidebearing | 70                         |
| Serif length        | ~2.6 × stroke              |
| Serif thickness     | ~0.6 × stroke (min 36)     |

Metrics deliberately match Wayfinder Sans so the two can be mixed in
multi-font CSS stacks without baseline drift.

## Weights

Same axis as Sans — Light through Black plus matching italics.
