# Redwood Serif — Design brief

An old-style / transitional serif with stroke contrast and bracketed
serifs. Elegant, slightly bookish, with the warmth of an early
20th-century Yosemite field journal.

## Inspiration

- Vicarel Studios' *John Muir Serif* (general genre and mood).
- Early-20th-century park-publication typography (park lodges'
  printed menus, ranger guidebooks, WPA poster body copy).

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 700                        |
| x-height            | 490                        |
| Ascender            | 800                        |
| Descender           | -200                       |
| Main stem           | 78                         |
| Hairline            | 36                         |
| Serif extension     | 34 (each side of stem)     |
| Serif height        | 34                         |
| Default sidebearing | 60                         |

## Character

- **Stroke contrast.** Thick stems ~78, thin curves/crossbars ~36.
- **Bracketed rectangular serifs** at the terminal ends of vertical
  stems (I, B, D, E, F, H, K, L, M, N, P, R, T, U, h, b, d, i, k, l,
  m, n, r, t, u).
- **Two-storey `a`** (single bowl + tall right stem).
- **Single-storey `g`** with descender hook.
- **Curled `Q` tail**, **flared `R` leg**.
- No italic in v1.
