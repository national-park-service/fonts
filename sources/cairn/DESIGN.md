# Cairn — Design brief

A bold geometric all-caps for signage, badges, and wayfinding. Two
weights. Lowercase glyphs reuse the uppercase outlines (small-caps
behavior is TODO).

## Targets

| Metric              | Value (em units, UPM 1000) |
| ------------------- | -------------------------- |
| Cap height          | 720                        |
| x-height            | 720 (== cap)               |
| Ascender            | 760                        |
| Descender           | -120                       |
| Default sidebearing | 80                         |
| Stroke (Regular)    | 140                        |
| Stroke (Bold)       | 180                        |

## Use

- Signage and trail markers (the namesake!)
- Badges, monograms, hero headlines
- Pairs naturally with Wayfinder Sans for body copy
