# @nps-fonts/wayfinder-sans

Wayfinder Sans — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **This project is independent and not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/wayfinder-sans
# or: npm install @nps-fonts/wayfinder-sans
```

## Use

```css
@import "@nps-fonts/wayfinder-sans";

body { font-family: "Wayfinder Sans", system-ui, sans-serif; }
```

Or via CDN, no install required:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/wayfinder-sans/index.css">
```

## Masters

- **Light** — weight 300
- **Light Italic** — weight 300, italic
- **Regular** — weight 400
- **Italic** — weight 400, italic
- **Medium** — weight 500
- **Medium Italic** — weight 500, italic
- **Bold** — weight 700
- **Bold Italic** — weight 700, italic
- **Black** — weight 900
- **Black Italic** — weight 900, italic
