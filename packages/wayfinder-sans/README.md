# @nps-fonts/wayfinder-sans

Wayfinder Sans — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Heritage

Forked from **[Big Shoulders Display](https://github.com/google/fonts/tree/main/ofl/bigshouldersdisplay)** by Patric King (House Industries) and the Big Shoulders Project Authors (OFL-1.1). Renamed and re-released under SIL OFL 1.1.

## Install

```bash
bun add @nps-fonts/wayfinder-sans
# or: npm install @nps-fonts/wayfinder-sans
```

## Use

```css
@import "@nps-fonts/wayfinder-sans";

body { font-family: "Wayfinder Sans", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/wayfinder-sans/index.css">
```
