# @nps-fonts/switchback

Switchback — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Heritage

Forked from **[Bowlby One](https://github.com/google/fonts/tree/main/ofl/bowlbyone)** by Vernon Adams (OFL-1.1). Renamed and re-released under SIL OFL 1.1.

## Install

```bash
bun add @nps-fonts/switchback
# or: npm install @nps-fonts/switchback
```

## Use

```css
@import "@nps-fonts/switchback";

body { font-family: "Switchback", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/switchback/index.css">
```
