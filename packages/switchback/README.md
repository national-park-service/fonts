# @nps-fonts/switchback

Switchback — Routed-trail display caps with Clean and Rough sibling cuts — backcountry signage.

Open-source typeface inspired by U.S. public-lands signage traditions, including National Park Service and National Forest wayfinding. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/switchback
# or: npm install @nps-fonts/switchback
```

## Use

```css
@import "@nps-fonts/switchback";

body { font-family: "Switchback", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/switchback/index.css">
```
