# @nps-fonts/switchback

Switchback — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **This project is independent and not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/switchback
# or: npm install @nps-fonts/switchback
```

## Use

```css
@import "@nps-fonts/switchback";

body { font-family: "Switchback", system-ui, sans-serif; }
```

Or via CDN, no install required:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/switchback/index.css">
```

## Masters

- **Regular** — weight 700
