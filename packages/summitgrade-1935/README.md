# @nps-fonts/summitgrade-1935

Summitgrade 1935 — Vintage 1930s NPS display caps — routed-redwood, CCC-era.

Original parametric font, drawn from scratch by NPS Fonts contributors. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/summitgrade-1935
# or: npm install @nps-fonts/summitgrade-1935
```

## Use

```css
@import "@nps-fonts/summitgrade-1935";

body { font-family: "Summitgrade 1935", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/summitgrade-1935/index.css">
```
