# @nps-fonts/redwood-serif

Redwood Serif — Old-style serif with bracketed serifs and stroke contrast — naturalist field-journal warmth.

Open-source typeface inspired by U.S. public-lands signage traditions, including National Park Service and National Forest wayfinding. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/redwood-serif
# or: npm install @nps-fonts/redwood-serif
```

## Use

```css
@import "@nps-fonts/redwood-serif";

body { font-family: "Redwood Serif", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/redwood-serif/index.css">
```
