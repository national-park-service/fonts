# @nps-fonts/cairn

Cairn — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Heritage

Forked from **[Public Sans](https://github.com/google/fonts/tree/main/ofl/publicsans)** by U.S. Web Design System (USWDS) and Public Sans Project Authors (OFL-1.1). Renamed and re-released under SIL OFL 1.1.

## Install

```bash
bun add @nps-fonts/cairn
# or: npm install @nps-fonts/cairn
```

## Use

```css
@import "@nps-fonts/cairn";

body { font-family: "Cairn", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/cairn/index.css">
```
