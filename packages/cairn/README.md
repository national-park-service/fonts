# @nps-fonts/cairn

Cairn — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **This project is independent and not affiliated with the U.S. National Park Service.**

## Install

```bash
bun add @nps-fonts/cairn
# or: npm install @nps-fonts/cairn
```

## Use

```css
@import "@nps-fonts/cairn";

body { font-family: "Cairn", system-ui, sans-serif; }
```

Or via CDN, no install required:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/cairn/index.css">
```

## Masters

- **Regular** — weight 400
- **Bold** — weight 700

## Files

| Format | Path |
|---|---|
| OTF    | `fonts/otf/Cairn-*.otf` |
| WOFF   | `fonts/woff/Cairn-*.woff` |
| WOFF2  | `fonts/woff2/Cairn-*.woff2` |
