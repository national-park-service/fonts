# @nps-fonts/wayfinder-serif

Wayfinder Serif — open-source typeface inspired by U.S. National Park Service signage. Released under the [SIL Open Font License 1.1](./LICENSE). **Independent project, not affiliated with the U.S. National Park Service.**

## Heritage

Forked from **[Zilla Slab](https://github.com/google/fonts/tree/main/ofl/zillaslab)** by Typotheque (commissioned by Mozilla) (OFL-1.1). Renamed and re-released under SIL OFL 1.1.

## Install

```bash
bun add @nps-fonts/wayfinder-serif
# or: npm install @nps-fonts/wayfinder-serif
```

## Use

```css
@import "@nps-fonts/wayfinder-serif";

body { font-family: "Wayfinder Serif", system-ui, sans-serif; }
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@nps-fonts/wayfinder-serif/index.css">
```
