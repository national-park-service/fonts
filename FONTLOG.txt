FONTLOG for NPS Fonts
=====================

This file provides detailed information on each font family in the
nps-fonts project, including the upstream sources we forked from and
the modifications we apply. Information is updated as releases ship.

PROJECT
-------
NPS Fonts — open-source typefaces inspired by U.S. National Park Service
signage. Independent project, not affiliated with the NPS.

Each family is a fork of an open-source typeface in the appropriate
genre, renamed to a Reserved Font Name we hold and re-released under
the SIL Open Font License v1.1. Original copyrights are preserved in
each binary; ours are appended.


CHANGELOG
---------

19 Apr 2026 (NPS Fonts contributors) — v0.2.0
- Pivoted to fork-based pipeline. Each family now derives from an
  established OFL-licensed source, renamed and repackaged.
- Per-family sources:
    Wayfinder Sans   ← Big Shoulders Display (Patric King, OFL)
                       — variable font, weight axis 100–900
    Wayfinder Serif  ← Zilla Slab (Typotheque / Mozilla, OFL)
                       — Light/Regular/Medium/Bold + matching italics
    Campfire Script  ← Caveat Brush (Pablo Impallari, OFL)
                       — single weight
    Switchback       ← Bowlby One (Vernon Adams, OFL)
                       — single weight
    Cairn            ← Public Sans (USWDS, OFL)
                       — variable font upright + italic, weight 100–900
- 13 masters across 5 families (~5 MB total, all formats).
- Outputs: .otf, .ttf, .woff, .woff2 per master.
- Each binary preserves the upstream copyright + license verbatim and
  appends ours per OFL §1. Reserved Font Names claimed for each family.
- Removed the v0.0.x / v0.1.x parametric drawing code. The fork pipeline
  produces type-designer-quality outlines that the parametric system
  could not approach.

19 Apr 2026 (NPS Fonts contributors) — v0.1.0
- Refined parametric outline construction, Latin-1 coverage, kerning,
  ligatures, italic overrides, polished specimen, expanded test suite.

18 Apr 2026 (NPS Fonts contributors) — v0.0.1
- Initial seed release with procedural outline placeholders.
- Build pipeline, CI/CD, web specimen, npm packaging established.


PER-FAMILY ATTRIBUTION
----------------------

### Wayfinder Sans
- Source: **Big Shoulders Display** by Patric King (House Industries)
  and the Big Shoulders Project Authors.
- Source license: SIL Open Font License v1.1.
- Source repository:
  https://github.com/google/fonts/tree/main/ofl/bigshouldersdisplay
- Original Reserved Font Name: "Big Shoulders Display".
- Our modifications: name-table rename to "Wayfinder Sans". Outlines
  unchanged.

### Wayfinder Serif
- Source: **Zilla Slab** by Typotheque, commissioned by Mozilla.
- Source license: SIL Open Font License v1.1.
- Source repository:
  https://github.com/google/fonts/tree/main/ofl/zillaslab
- Original Reserved Font Name: "Zilla Slab".
- Our modifications: name-table rename to "Wayfinder Serif". Outlines
  unchanged.

### Campfire Script
- Source: **Caveat Brush** by Pablo Impallari (Impallari Type).
- Source license: SIL Open Font License v1.1.
- Source repository:
  https://github.com/google/fonts/tree/main/ofl/caveatbrush
- Original Reserved Font Name: "Caveat Brush".
- Our modifications: name-table rename to "Campfire Script". Outlines
  unchanged.

### Switchback
- Source: **Bowlby One** by Vernon Adams.
- Source license: SIL Open Font License v1.1.
- Source repository:
  https://github.com/google/fonts/tree/main/ofl/bowlbyone
- Original Reserved Font Name: "Bowlby One".
- Our modifications: name-table rename to "Switchback". Outlines
  unchanged.

### Cairn
- Source: **Public Sans** by the U.S. Web Design System and Public Sans
  Project Authors.
- Source license: SIL Open Font License v1.1.
- Source repository:
  https://github.com/google/fonts/tree/main/ofl/publicsans
- Original Reserved Font Name: "Public Sans".
- Our modifications: name-table rename to "Cairn". Outlines unchanged.
